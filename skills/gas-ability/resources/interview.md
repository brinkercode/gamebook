# GAS Ability — Scoping Questions

1. **Ability name and purpose?** (e.g., "Sprint", "ThrowGrenade", "Dash", "HealSelf", "ADS")
2. **Activation type?**
   - Instant: fires once on button press (ThrowGrenade, Heal)
   - Duration: active while button held (Sprint, ADS)
   - Passive: always active, no input (HealthRegen, ArmorBuff)
3. **Is there a resource cost?** (Stamina / Ammo / SpecialCharge — specify attribute name and amount)
4. **Is there a cooldown?** (seconds; 0 = no cooldown)
